# Hermes Games

A decentralized gaming platform built with React, TypeScript, and Vite.

## Available Games

Based on the project structure, the following game components are available:

- CoinFlip
- Crash
- Dice
- Mines
- Plinko
- Roulette

## Development

1.  Install dependencies:
    ```bash
    yarn install
    ```
2.  Start the development server:
    ```bash
    yarn dev
    ```
